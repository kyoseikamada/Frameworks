経営戦略
BSC — バランスト・スコアカード
exec-01
財務だけでなく4つの視点でバランスよく戦略を評価・管理する（Kaplan & Norton, 1992）
財
財務の視点
株主にどう見られたいか？
売上成長率 / 利益率 / ROI / EVA / コスト削減
顧
顧客の視点
顧客にどう見られたいか？
顧客満足度 / 市場シェア / NPS / 顧客維持率
業
業務プロセスの視点
何を優れて行うべきか？
プロセス効率 / 品質コスト / サイクルタイム / 不良率
学
学習・成長の視点
改善・革新し続けられるか？
従業員満足度 / スキル習得率 / 情報システム整備度
戦略マップ： 4つの視点を「学習・成長 → 業務プロセス → 顧客 → 財務」の因果連鎖で繋ぎ、戦略の論理を可視化する。
