標準入出力
名称FDデフォルト
標準入力 (stdin)0キーボード
標準出力 (stdout)1ターミナル
標準エラー出力 (stderr)2ターミナル
リダイレクト記号
記号意味
> filestdout → ファイル（上書き）
>> filestdout → ファイル（追記）
2> filestderr → ファイル
2>&1stderrをstdoutに合流
&> filestdout+stderr → ファイル
< fileファイル → stdin
<<EOFヒアドキュメント
実例
ls -la > list.txt
echo "append" >> log.txt

# stderr を捨てる
find / -name "*.conf" 2>/dev/null

# stdout+stderr をまとめて保存
./app.sh &> output.log

# パイプ
cat /etc/passwd | grep alice
ls -la | sort -k5 -n | tail -10
ps aux | grep java | grep -v grep

# コマンド置換
TODAY=$(date +%Y%m%d)
echo "Today is $TODAY"

# tee: ファイルに保存しつつ画面にも出力
make 2>&1 | tee build.log

# xargs: stdoutを引数として渡す
find . -name "*.log" | xargs gzip

# プロセス置換
diff <(sort file1.txt) <(sort file2.txt)
