主要シグナル
番号名前意味
1SIGHUP設定再読み込み（nginxなど）
2SIGINTCtrl+C（割り込み）
9SIGKILL強制終了（キャッチ不可）
15SIGTERM正常終了要求（デフォルト）
19SIGSTOP一時停止（キャッチ不可）
プロセス操作コマンド
# プロセス一覧
ps aux
ps -ef --forest  # ツリー表示

# リアルタイム監視
top
htop             # 色付きで見やすい

# シグナル送信
kill 1234         # SIGTERM（正常終了）
kill -9 1234      # SIGKILL（強制終了）
kill -HUP 1234    # SIGHUP（設定再読込）
pkill nginx       # 名前でkill
killall java      # 同名全プロセスkill

# バックグラウンド実行
./script.sh &
nohup ./script.sh &  # ログアウト後も継続

# ジョブ管理
jobs              # ジョブ一覧
fg %1             # フォアグラウンドに戻す
bg %1             # バックグラウンドで再開
Ctrl+Z            # 一時停止
kill -9 は後処理（ファイルクローズ・ロック解放）なしで強制終了するため、最終手段として使う。まず kill（SIGTERM）を試すこと。
