アプリケーションサーバー
Jakarta EE は仕様。アプリサーバーが実装を提供する。WAR / EAR 形式でデプロイされ、サーバーがコンテナとして動作する。
サーバー種別
WildFly（旧 JBoss）OSS / Red Hat
GlassFishOSS / リファレンス実装
PayaraGlassFish 派生・商用サポートあり
Open LibertyOSS / IBM
WebLogic商用 / Oracle
WebSphere商用 / IBM
⚠ Tomcat はサーブレットコンテナのみ（EJB・JPA 等は含まない）。Spring Boot + Tomcat 組み合わせが実務では最も多い。
