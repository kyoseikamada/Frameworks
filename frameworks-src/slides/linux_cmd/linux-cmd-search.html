Linuxコマンド
ファイル検索（find）
linux-cmd-03
find — ファイルを探す
# 基本形: find 探す場所 条件
find /etc -name "*.conf"
# → /etc 以下の .conf ファイルを全て表示

# カレントディレクトリから探す
find . -name "Main.java"

# 名前の一部で探す（* はワイルドカード）
find . -name "*test*"

# ファイルだけ、フォルダだけに絞る
find /tmp -type f    # ファイルのみ
find /var -type d    # フォルダのみ

# 更新日時で絞る
find /tmp -mtime -1  # 1日以内に更新されたもの
エラーを非表示にする
# "Permission denied" が大量に出る場合
find / -name "*.conf" 2>/dev/null
# 2>/dev/null = エラー出力を捨てる
which / whereis — コマンドの場所を調べる
# このコマンドはどこにある？
which python3
# → /usr/bin/python3

which git
# → /usr/bin/git

# コマンドが入っているか確認にも使える
which docker || echo "docker is not installed"
よく使う検索パターン
# 設定ファイルを探す
find /etc -name "nginx.conf"

# ログファイルを探す
find /var/log -name "*.log"

# 特定の拡張子のファイルを全て探す
find . -name "*.java" | wc -l  # 件数確認
find . -name "*.java"          # 一覧表示

# 今日更新されたファイルを探す
find . -mtime -1 -type f
よくある間違い：find -name *.conf のように "" を省略するとシェルがワイルドカードを展開してしまう。必ず "*.conf" のようにクォートで囲む。
