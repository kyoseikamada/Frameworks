linux-cmd-05
ネットワークの基本確認
# 自分のIPアドレスを確認する
ip addr show
hostname -I        # IPアドレスだけ表示

# 接続できるか確認する（ping）
ping google.com
ping -c 3 8.8.8.8  # 3回だけ送って終わる
# Ctrl+C で終了

# URLにアクセスする（curl）
curl https://example.com           # 本文を表示
curl -I https://example.com        # ヘッダーだけ確認
curl -o file.zip https://example.com/file.zip  # ダウンロード

# ファイルをダウンロードする（wget）
wget https://example.com/file.zip
SSH — リモートサーバーに接続する
# 基本的な接続
ssh ユーザー名@サーバーのIPアドレス
ssh alice@192.168.1.10

# 秘密鍵（.pem ファイル）を使って接続
ssh -i ~/.ssh/mykey.pem alice@192.168.1.10

# 接続を切る
exit
SCP — ファイルを転送する
# ローカル → リモート（アップロード）
scp memo.txt alice@192.168.1.10:/home/alice/

# リモート → ローカル（ダウンロード）
scp alice@192.168.1.10:/home/alice/memo.txt .

# フォルダごと転送（-r）
scp -r ./project alice@192.168.1.10:~/
初めてSSH接続するとき："Are you sure you want to continue connecting?" と聞かれたら yes と入力する。これはサーバーの公開鍵をローカルに登録する確認。
