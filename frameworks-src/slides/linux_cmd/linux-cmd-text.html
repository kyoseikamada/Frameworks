Linuxコマンド
テキスト表示・検索
linux-cmd-02
ファイルの中身を見る
# 全部表示する
cat memo.txt

# スクロールして読む（q で終了）
less app.log

# 先頭・末尾だけ表示
head -5 app.log     # 先頭5行
tail -5 app.log     # 末尾5行

# ログをリアルタイムで監視する
tail -f /var/log/syslog
# → 新しい行が書き込まれるたびに表示される
# → Ctrl+C で終了
行数・文字数を数える
wc -l app.log    # 行数を数える
wc -c app.log    # バイト数を数える
文字列を検索する（grep）
# ファイルの中から行を探す
grep "error" app.log

# 大文字・小文字を区別しない
grep -i "ERROR" app.log

# 行番号も表示する
grep -n "error" app.log

# 一致しない行だけ表示（除外）
grep -v "DEBUG" app.log

# フォルダの中を再帰的に検索
grep -r "TODO" ./src/
パイプ — コマンドをつなぐ
# "|" で前のコマンドの結果を次に渡す
cat /etc/passwd | grep alice
ls -la | grep ".log"

# エラーログだけ取り出して行数を数える
grep "ERROR" app.log | wc -l
less の操作：スペース=次ページ / b=前ページ / /キーワード=検索 / q=終了
