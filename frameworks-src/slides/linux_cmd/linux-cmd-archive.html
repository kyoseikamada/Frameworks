linux-cmd-06
ファイルを圧縮・展開する（tar）
# フォルダをまとめて圧縮する
tar czvf archive.tar.gz フォルダ名/
# c=作る z=gzip圧縮 v=進捗表示 f=ファイル名

# 圧縮ファイルを展開する
tar xzvf archive.tar.gz
# x=展開 z=gzip解凍 v=進捗表示 f=ファイル名

# 中身を確認する（展開せずに）
tar tzvf archive.tar.gz

# zipファイルの場合
zip -r archive.zip フォルダ名/
unzip archive.zip
覚え方：作るときは c（create）、展開するときは x（extract）。z は gzip 圧縮の有無。
ソフトをインストールする（apt）
# パッケージ情報を最新にする（まずこれ）
sudo apt update

# ソフトをインストール
sudo apt install git
sudo apt install nginx
sudo apt install vim

# ソフトを削除
sudo apt remove nginx

# インストール済みか確認
apt list --installed | grep git

# ソフトを検索する
apt search nginx
よくある流れ
# 1. まずパッケージ情報を更新
sudo apt update

# 2. インストール
sudo apt install docker.io

# 3. 自動起動を設定
sudo systemctl enable docker
sudo systemctl start docker

# 4. 動作確認
docker --version
